# Project Overview

This project was completed as part of the Digital Literacy (CSE0001) course and is aimed at developing practical digital competencies. It includes activities such as designing presentations, building a professional online portfolio, exploring coding and collaboration platforms, understanding email and social media etiquette, and learning about cyber safety. Through these modules, I gained hands-on experience with various digital tools and understood the importance of maintaining a responsible and professional online identity.
